Memorizer

This application can be used as a flashcard. Because there is sophisticated service such as Quizlet, this is for study only mainly for practice of React.

How to use

  1. Register flashcard from "+ Add" on Flashcard screen
  2. Register sentence from "+ Add" on Sentence screen
  3. Select flashcard to add from clip icon on Sentence screen
  4. Start playing by pressing play button from Flashcard screen
